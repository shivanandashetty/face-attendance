# 🎯 AI Face Recognition Attendance System

A full-stack AI-based Face Recognition Attendance System built using:

- ⚛️ React.js (Frontend)
- 🐍 Python Flask (Backend)
- 🧠 InsightFace + OpenCV (Face Recognition)
- 🗃️ SQLite (Database)

---

## 🚀 Features

- 👤 Add new persons
- 📸 Capture dataset using camera
- 🧠 Train AI model
- 🎥 Live camera face recognition
- 📝 Automatic attendance logging
- 🖼️ Gallery of captured faces
- 📊 Attendance reports dashboard

---

## 🗂️ Project Structure

face-attendance/
├── FACE/ # Backend (Flask + AI)
└── face-frontend/ # Frontend (React)



---

## ⚙️ How To Run Backend

cd FACE
python -m venv faceenv
faceenv\Scripts\activate
pip install -r requirements.txt
python api_server.py


---

## ⚛️ How To Run Frontend

cd face-frontend
npm install
npm start


---

## 🧠 Workflow

1. Create person
2. Register face (dataset capture)
3. Train model
4. Start live camera
5. Attendance is marked automatically
